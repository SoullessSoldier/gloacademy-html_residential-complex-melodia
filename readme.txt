День 4
Выбор квартир
Напишем логику работы всплывающего окна, 
которое появляется для выбора квартиры. 
Сделаем это с помощью CSS и JS. 

ЗАДАНИЕ


Напишите стили для всплывающего окна с выбором квартир. 
Это окно должно появляться при клике на этаж и при клике 
на кнопку «смотреть квартиры на этаже». 
Добавьте анимацию. Опубликуйте сайт в интернете и прикрепите ссылку на работающий сайт.

ДОПОЛНИТЕЛЬНОЕ ЗАДАНИЕ

При наведении на название квартиры в списке справа, 
должна подсвечиваться эта квартира на схеме слева и наоборот. 
При наведении на квартиру на схеме, должна подчеркиваться ссылка на 
эту квартиру справа. Сделать это можно по аналогии со схемой этажей.
